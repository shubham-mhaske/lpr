Approach Details :

Details about the system are mentioned below :

User End


1.   User will get an system to add and delete vehicles from database.
2.   User can get notifications about unknown vehicles entering their premises.
3.   User will get to track movement of vehicles.

Internal Processing

1.   The video given as an input will be processed and we will get details about vehicle license plate.
2.   The number on the license plate will be checked with numbers in database.
3.   If the vehicle is unknown notify it to the users.
